136. 只出现一次的数字

发表于 2018-09-14 | 更新于 2018-12-26 | 分类于 leetCode题库笔记 | | 阅读次数：
本文字数： 1.6k | 阅读时长 ≈ 1 分钟

#一、题目原型：

给定一个非空整数数组，除了某个元素只出现一次以外，其余每个元素均出现两次。找出那个只出现了一次的元素

#二、题目意思剖析：

#三、解题思路：

######第一种
根据题目意思，数组的个数一定是奇数。(2+2+2+1+2)类似这样,先排个序，然后 i = i + 2，一旦前后不同，就return前面那个数字。

######第二种：异或法
找到数组里唯一不同的那个数字，其实可以用

{
var temp: Int = 2
temp ^= 2
print(temp)
}
打印的temp = 0

{
var temp: Int = 0
temp ^= 2
print(temp)
}
打印的temp = 2

func singleNumber(_ nums: [Int]) -> Int {
if nums.count == 1 {
return nums.first!
}
// 异或
var temp: Int = 0
for num in nums {
temp ^= num
}
return temp
}
`

#四、小结
